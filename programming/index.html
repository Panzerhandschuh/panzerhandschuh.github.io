C++

FBX Model Converter

The FBX model converter is a tool that uses Autodesk's FBX SDK to convert FBX files to a custom binary format for my game Protowave. I created the binary format for Protowave so that models can be loaded at runtime very quickly. The tool also generates material properties so that Unity can read the shader and shader attributes used on each material. This tool will allow users to import their own models into the game, allowing a much greater level of customization for the track editor.

C#

Realtime Path Deformation

For Protowave's track editor, I had to create an algorithm to deform an arbitrary mesh along a path. I did this be forming a bezier curve from two points, and aligning the mesh's vertices to fit the curve. The algorithm was made more efficient for runtime operation by grouping the mesh's vertices into cross sections with a custom data structure. Expensive calculations would only have to be done on each cross section of the mesh instead of on every vertex.
