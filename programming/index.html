FBX Model Converter

The FBX model converter is a tool that uses Autodesk's FBX SDK to convert FBX files to a custom binary format for my game Protowave. I created the binary format for Protowave so that models can be loaded at runtime very quickly. A custom asset management system also had to be created to handle references to imported assets. When all references to an asset are removed, the asset would be freed from memory. The tool also generates material properties so that Unity can read the shader attributes used on each material. This tool will allow users to import their own models into the game, allowing for a much greater level of customization for the track editor.

Java

Distributed Instant Essay Grading System

For this project, I coded a Java service that allows short answer questions to be graded nearly instantly. This was done using Apache Hadoop Distributed File System (HDFS) to share grader files among nodes in a server cluster and Apache Zookeeper to route and load balance requests. Additionally, sockets were used to communicate between a master node and worker nodes in a hierarchical fashion. The master node handles all incoming requests from a web app and then routes the requests to be graded on a worker node with the lowest traffic.
