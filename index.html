Projects

These are some of the bigger projects that I have worked on in my spare time.

Protowave

Protowave is a futuristic racing game with an advanced track editor created with Unity engine. The track editor allows users to create and bend tracks into any path that they desire. The game also features high speed, intense gameplay similar to games of the F-Zero franchise.

Blitzkrieg Assault

This is a map created with Starcraft 2's Galaxy Editor. The objective of this map is to create unit spawning buildings to fight a hoard of enemy units. There are player controlled heroes and challenging boss fights. This map received much success after I created a promotional video and featured it on the website, SC2Mapster.

Eternal

Eternal is a map created with Hammer for Counter-Strike: Source. It uses a special gamemode called "surf" where players slide on ramps until they reach the end of a course. Much care was taken in balancing the layout of the map to create a solid experience for players. I have also invested a lot of time detailing the map to create an interesting and unique fantasy theme.
