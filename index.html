Projects

Protowave (In Development)

Protowave is a futuristic racing game with an advanced track editor created with Unity engine. The track editor allows users to create and manipulate tracks with bezier curves. The game also features high speed, intense gameplay similar to games of the F-Zero franchise.

Blitzkrieg Assault (Download)

This is a map created with Starcraft 2's Galaxy Editor. The objective of this map is to create unit spawning buildings to fight a hoard of enemy units. There are player controlled heroes and challenging boss fights. This map received much popularity after I created a promotional video and featured it on the website, SC2Mapster.

Eternal (Download)

Eternal is a map created with Hammer for Counter-Strike: Source. It uses a special gamemode called "surf" where players slide on ramps until they reach the end of a course. Much care was taken in balancing the layout of the map to create a solid experience for players. I have also invested a lot of time detailing the map to create an interesting and unique fantasy theme.
